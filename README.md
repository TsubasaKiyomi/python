# リスト型

## リスト型とは
- 複数の要素が順序を持って並べられているデータ型のこと

- []ブランケットで囲む。文字、数字どちら可能

- シーケンス型の一つがリスト　複数の要素が順序を持って並べられているデータ型のこと

- ミュータブルで要素の*削除や挿入ができる。*
#### ミュータブルとは
- 変更可能なオブジェクトのこと

## リスト型の使い方
```
l = [1,2,3,4,5]

print = (l)

#[1,2,3,4,5]

l = [1,2,3,4,5]

print = (l[2])

#3

- ミュータブルで要素を挿入

l = ['a', 'b', 'c', 'd', 'e']

l[2] = 'z'

print(l)

#a,b,z,d,e
```
# タプル型

## タプル型とは
- 複数のデータを一直線に並べたタイプのデータ型

- リストと違い変更、削除ができない。イミュータブル（ immutable ）
- ()で囲む or 「,」カンマをつけるとタプルになる
- リスト[]と辞書{}と併用できる

#### イミュータブル（ immutable ）

「厳密にはオブジェクトid を変えずに、要素の追加、変更、削除ができない」という意味。

## タプル型の使い方
```
t = (1,2,3,4,5,6,1,2,3,4)

t[0] # 1

t[-1] # 4

t[5:] # 6,1,2,3,4

t[0]=200 #Errorになる。変更、追加、削除出来ないから。

t=([1,2,3][4,5,6]) タプルにリストを入れることができる。

t[0][0]=100

#([100,2,3][4,5,6])リストに入れるのでErrorにはならない。

t=(1,2,3) + (4,5,6) 

#(1,2,3,4,5,6)タプル同士を繋げて新しいタプルを作れる。
#### タプルのアンパック
- 複数の要素を持つ物を分解して各変数に代入すること。(展開すること)
#### タプルのアンパック使い方
t =10,20

print(t) #10,20

x,y=t

print(x,y) #10,20

a=100
b=200
#100,200
- aとbの中を入れ替える。
a,b = b,a #200,100
```

# while文

## whileとは
- 条件を指定し”条件が真の間は繰り返し処理が実行”できる。
- while文は「条件を指定し」その条件が"False"になればループを抜ける。つまり、条件がTrueの間、ループ処理を続ける。

## while_break_continue_ の使い方
```
count = 0
while True:
    if count > = 5:
        break

    elif count ==3:
          count += 1
          continue
else:
    print('done')

```
- break（ループから抜ける）
- continue(スキップ)
- elseは、whileループ中にbreakしなければelse('done')が出力される。

# for...else

## for...else とは

for 文のブロックを実行し、その後 else ブロックが実行される。。

breakしなければ、elseのprint()が出力される。

## for...else の使い方

```
for i in range(10):
    print(i)

else:
    print("else is called")

```
