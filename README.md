# リスト型

## リスト型とは

- 複数の要素を含むコンテナ型のオブジェクト

- []ブランケットで囲む。文字、数字どちら可能

- ミュータブルで要素の*削除や挿入ができる。*
#### ミュータブルとは
- 変更可能なオブジェクトのこと

## リスト型の使い方

l = [1,2,3,4,5]

print = (l)

#[1,2,3,4,5]

l = [1,2,3,4,5]

print = (l[2])

#3

- ミュータブルで要素を挿入

l = ['a', 'b', 'c', 'd', 'e']

l[2] = 'z'

print(l)

#a,b,z,d,e

# タプル型

## タプル型とは
- リストと違い変更、削除ができない。イミュータブル（ immutable ）
- ()で囲む or 「,」カンマをつけるとタプルになる
- リスト[]と辞書{}と併用できる

#### イミュータブル（ immutable ）

「厳密にはオブジェクトid を変えずに、要素の追加、変更、削除ができない」という意味。

## タプル型の使い方
t = (1,2,3,4,5,6,1,2,3,4)

t[0] # 1

t[-1] # 4

t[5:] # 6,1,2,3,4

t[0]=200 #Errorになる。変更、追加、削除出来ないから。

t=([1,2,3][4,5,6]) タプルにリストを入れることができる。

t[0][0]=100

#([100,2,3][4,5,6])リストに入れるのでErrorにはならない。

t=(1,2,3) + (4,5,6) 

#(1,2,3,4,5,6)タプル同士を繋げて新しいタプルを作れる。
#### タプルのアンパック
- 複数の要素を持つ物を分解して各変数に代入すること。(展開すること)
#### タプルのアンパック使い方
t =10,20

print(t) #10,20

x,y=t

print(x,y) #10,20

a=100
b=200
#100,200
- aとbの中を入れ替える。
a,b = b,a #200,100
